### Steam自建服

服务器名称：米奇不妙屋  
密码：789456  
平台：Steam  
世界：森林、洞穴、海滩、火山  
开放时间：7x24H

简介：
这是当前主要游玩的 Steam 饥荒联机版服务器。服务器包含地上、洞穴、海难、火山四个世界，已开启基础功能模组与少量内容扩展模组。房间 Mod 会按稳定性和平衡性调整，旧版长列表不再直接展示，避免误导玩家按过期列表订阅。

#### 怎么加入

方法一：在游戏内搜索房间名 `米奇不妙屋`，找到房间后输入密码 `789456` 加入。

方法二：在游戏中按下 ` 键（Esc 下方的键）调出控制台，输入：

```lua
c_connect("games.homura.top", 10999, "789456")
```

如果域名暂时无法解析，可以联系服主获取当前直连地址。

#### 服务器说明

- 当前服务器为四世界：森林 / 洞穴 / 海滩 / 火山。
- 房间会根据崩溃、卡顿、平衡性随时调整 Mod，不建议照旧文章里的历史 Mod 列表逐个订阅。
- 玩家一般只需要正常进房，缺少的创意工坊内容会由游戏自动订阅/下载。
- 如果进入海难相关世界时崩溃，优先安装下面的服务器专用补丁。

#### 服务器专用 Mod / 补丁

当前服务器使用了一个针对 Island Adventures Core 的补丁包，用于修复进入海难世界时的闪退问题。

- 补丁对象：`3435352667` / Island Adventures - Core / 岛屿冒险 - 核心
- 触发问题的 Mod：`1467214795` / Island Adventures - Shipwrecked / 岛屿冒险 - 海难
- 典型报错：`SetIAOverhang nil`、`SetBaseTile nil`、`RunWithoutIACorners nil`
- 补丁作用：让核心 Mod 加载海难世界需要的地图方法。

下载：

- [3435352667_IA_Core_patched_20260704035721.zip](/assets/dst_mod_backups/3435352667_IA_Core_patched_20260704035721.zip)
- [补丁安装说明 INSTALL_IA_CORE_PATCH.txt](/assets/dst_mod_backups/INSTALL_IA_CORE_PATCH.txt)

客户端安装简要步骤：

1. 完全退出《Don't Starve Together》。
2. 找到 Steam Workshop 本地目录，常见路径：

   ```text
   E:\SteamLibrary\steamapps\workshop\content\322330\3435352667
   C:\Program Files (x86)\Steam\steamapps\workshop\content\322330\3435352667
   ```

3. 备份原目录中的：

   ```text
   3435352667\main\postinit.lua
   ```

4. 用补丁包里的 `3435352667` 覆盖原目录，或只替换：

   ```text
   3435352667\main\postinit.lua
   ```

5. 重新启动游戏后再加入服务器。
